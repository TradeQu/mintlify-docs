## Contact

- **General support:** [support@meshqu.com](mailto:support@meshqu.com)
- **Security issues:** [security@meshqu.com](mailto:security@meshqu.com)
- **Early access requests:** [contact@meshqu.com](mailto:contact@meshqu.com)

## When contacting support

Include the `correlation_id` from the API response. This is returned in every response body and as the `x-correlation-id` header. It allows us to trace your request immediately.

## Responsible disclosure

If you discover a security vulnerability, contact [security@meshqu.com](mailto:security@meshqu.com). Do not disclose publicly until a fix is available.
